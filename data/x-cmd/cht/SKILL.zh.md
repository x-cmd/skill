# cht - 开发者速查表与代码片段

`cht` 模块利用 cheat.sh 平台提供高质量的、经社区验证的代码片段和命令使用示例。它专为需要即时获取“如何做”答案的开发者而设计。

## 激活时机
- 当用户需要特定编程语言的代码示例时（例如 `python/hello`）。
- 当寻求命令的简洁“速查表”时（例如 `ls`）。
- 当用户想要使用 “learn-x-in-minutes” 内容快速学习一门语言时 (`--learn`)。
- 当使用关键词在多个速查表中进行搜索时。

## 核心原则与规范
- **示例驱动**: 专注于提供可运行的代码片段或直接的命令示例。
- **内容广度**: 涵盖数百种 CLI 工具和数十种编程语言。
- **搜索能力**: 使用 `-s` 在 cheat.sh 生态系统中进行基于关键词的搜索。

## 实战示例

### 语言代码片段
```bash
# 从 cheat.sh 获取 Python 的 'hello world' 示例
x cht python/hello
```

### 命令速查表
```bash
# 查看 'sed' 常用模式的简洁摘要
x cht sed
```

### 快速学习
```bash
# 显示 Go 语言的初学者速查表
x cht --learn go
```
