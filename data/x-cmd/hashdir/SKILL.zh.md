# hashdir - 递归目录哈希工具

`hashdir` 模块递归地为目录树中的所有文件计算哈希，生成校验和与相对路径的合并列表，用于完整性验证。

## 激活时机
- 当比较两个目录结构在内容上的差异时。
- 当为特定文件夹（如配置或源代码）创建安全基准时。
- 当通过基准哈希文件在版本控制中跟踪目录更改时。

## 核心原则与规范
- **一致性**: 文件按字母顺序排序，以确保无论操作系统遍历顺序如何，都能获得一致的哈希输出。
- **相对路径**: 在输出中使用相对路径，以便在不同的根路径之间实现可移植性。

## 实战示例

### 目录完整性
```bash
# 为项目目录计算 SHA256 哈希列表
x hashdir --sha256 ./my_project
```

### 基准比较
```bash
# 将当前目录状态与保存的 MD5 基准进行比较
x hashdir . | diff - baseline.md5
```
