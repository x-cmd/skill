# ncdu - 交互式磁盘使用情况分析器

`ncdu` 模块提供了一个可视化磁盘空间使用情况的交互式界面。它通过 x-cmd 包管理器下载并执行，确保用户即使本地未安装该工具也能使用。

## 激活时机
- 当调查哪些文件和文件夹消耗了最多的磁盘空间时。
- 当在交互式的、基于终端的 UI 中执行磁盘清理时。
- 当在受限环境中需要零设置的磁盘使用分析器时。

## 核心原则与规范
- **交互式清理**: 允许直接浏览目录树并删除大型项目。
- **零设置**: 如果二进制文件缺失，会自动处理下载和执行。
- **参数传递**: 通过 `--` 分隔符支持所有标准 `ncdu` 标志（如 `--si` 或 `--exclude`）。

## 实战示例

### 分析当前目录
```bash
# 为当前目录启动交互式磁盘使用情况分析器
x ncdu
```

### 导出结果
```bash
# 扫描并将结果保存到文件中，以便稍后查看
x ncdu -- -o scan_results.txt
```
