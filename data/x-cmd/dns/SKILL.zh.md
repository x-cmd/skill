# dns - DNS 配置与管理

`dns` 模块提供了一个简单的界面，用于查看当前 DNS 设置、列出可用的 DNS 服务器，并执行维护任务（如刷新系统的 DNS 缓存）。

## 激活时机
- 当检查系统当前正在使用哪些 DNS 服务器时。
- 当排除域名解析故障时。
- 当需要刷新本地 DNS 缓存以应用更改时。

## 核心原则与规范
- **可见性**: 将 DNS 信息合并到单个命令 (`current`) 中。
- **维护**: 使用 `refresh` 清除缓存的名称解析数据。

## 实战示例

### 查看配置
```bash
# 显示当前系统 DNS 设置
x dns current
```

### 刷新缓存
```bash
# 刷新系统 DNS 缓存
x dns refresh
```

### 列出可用服务器
```bash
# 列出所有已配置的 DNS 服务器地址
x dns ls
```
