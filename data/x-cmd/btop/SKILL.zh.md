# btop - 系统资源监控器

`btop` 是一款高性能的系统监控器，实时显示 CPU、内存、磁盘、网络和进程的统计信息。它具有预设视图和交互式过滤功能，用于深度的系统分析。

## 激活时机
- 当用户想要系统资源使用的详细、视觉化细目时。
- 当使用以进程为中心的视图执行内存分析时。
- 当基于 PID 或名称以交互方式过滤和管理进程时。
- 当需要一个无需复杂设置即可在 Linux, macOS 和 Windows 上运行的资源监控器时。

## 核心原则与规范
- **预设视图**: 利用预设 (`-p`) 来关注不同的资源（例如，`-p 1` 用于内存/进程）。
- **交互式控件**: 利用快捷键，如 `m`（按内存排序）、`c`（按 CPU 排序）和 `/`（过滤），进行快速导航。
- **零依赖运行**: 如果本地未找到，会自动通过 `pixi` 或 `scoop` 下载并运行。

## 实战示例

### 默认视图
```bash
# 启动 btop 显示所有系统资源
x btop
```

### 以内存为中心的视图
```bash
# 以进程为中心的预设启动，便于内存分析
x btop -p 1
```

### 以 CPU 为中心的预设
```bash
# 以 CPU 性能为主要焦点启动
x btop -- --preset 2
```
