# sb - SBOM 生成与分析器

`sb` 模块提供了处理软件物料清单 (SBOM) 的工具，使用户能够生成项目组件的详细列表并执行以安全为中心的分析。

## 激活时机
- 当为项目生成 SBOM 报告时（例如使用 `syft` 或 `trivy` 后端）。
- 当为了安全合规审计软件依赖项时。
- 当在不同的 SBOM 格式之间进行转换时。

## 核心原则与规范
- **自动化**: 旨在集成到 CI/CD 流水线中，以实现自动化的 SBOM 生成。
- **后端支持**: 在底层利用流行的 SBOM 工具。

## 实战示例

### 生成 SBOM
```bash
# 为当前项目创建一个 SBOM 报告
x sb gen
```

### 查看组件列表
```bash
# 以人类可读的格式列出所有软件组件
x sb ls
```
