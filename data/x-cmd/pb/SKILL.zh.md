# pb - 跨平台剪贴板工具

`pb` 模块为 macOS、Windows、Linux 和 Android 的系统剪贴板提供了统一的界面。它会自动检测环境，甚至支持通过 OSC52 协议在 SSH 会话中进行复制。

## 激活时机
- 当用户想要将命令输出或文本复制到系统剪贴板时。
- 当将剪贴板内容粘贴到终端或命令中时。
- 当在远程 SSH 会话中工作，并需要将远程剪贴板与本地机器同步时。

## 核心原则与规范
- **环境无关**: 一个命令 (`x pb`) 即可取代 `pbcopy`, `xclip`, `wl-copy` 和 `clip.exe`。
- **远程同步**: 在 SSH 会话中自动使用 OSC52，允许将内容从服务器复制到本地客户端。
- **输入灵活性**: 同时支持管道输入和直接参数输入进行复制。

## 实战示例

### 从管道复制
```bash
# 将命令结果复制到剪贴板
echo "要复制的数据" | x pb
```

### 复制参数内容
```bash
# 直接复制特定文本
x pb copy "你好世界"
```

### 粘贴到终端
```bash
# 粘贴当前剪贴板内容
x pb paste
```

## 交付验证清单
- [ ] 确认操作是复制还是粘贴。
- [ ] 如果在 Linux 上，确保系统已安装剪贴板管理器 (xclip/wl-copy)。
- [ ] 如果远程工作，验证 SSH 客户端是否支持 OSC52。
