# wttr - 命令行天气预报

`wttr` 模块利用 wttr.in 服务为当前或指定位置提供天气信息。它支持人机可读格式、月相和单位自定义。

## 激活时机
- 当用户想要检查当前位置或特定城市的天气时。
- 当查看当前或历史月相时。
- 当为脚本或状态栏需要简洁的天气摘要时。

## 核心原则与规范
- **本地化**: 除非特别指定，否则根据环境自动调整单位和语言。
- **月相**: 为月球数据使用 `moon` 子命令。

## 实战示例

### 检查本地天气
```bash
# 查看当前位置的天气预报
x wttr
```

### 检查特定城市
```bash
# 查看北京的天气
x wttr Beijing
```

### 月相
```bash
# 查看当前的月相
x wttr moon
```
