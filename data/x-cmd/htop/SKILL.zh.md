# htop - 经典的进程与资源监控器

`htop` 模块通过利用 `pixi` 包管理器，确保用户始终可以访问经典的交互式进程查看器，即使是在未预装它的环境中。

## 激活时机
- 当用户想要以交互方式查看和管理运行中的进程时。
- 当执行 CPU、内存和交换分区使用的实时监控时。
- 当需要在不进行全系统更改的情况下快速安装并执行 `htop` 时。

## 核心原则与规范
- **环境无关**: 在缺少 `htop` 的环境中使用 `pixi` 提供该工具。
- **完整参数支持**: 通过 `--` 分隔符支持所有标准的 `htop` 标志和选项。

## 实战示例

### 标准监控
```bash
# 启动交互式进程监控器
x htop
```

### 过滤视图
```bash
# 监控特定用户的进程
x htop -- -u root
```

### 帮助与版本
```bash
# 访问 htop 内部帮助
x htop -- -h
```
