# gcal - GNU gcal 增强工具

`gcal` 模块为 GNU gcal 提供了一个强大的包装器，可以轻松显示具有本地化节假日支持和可自定义格式的日历。

## 激活时机
- 当用户想要打印月历或年历时。
- 当需要特定国家（如中国）的节假日信息时。
- 当初始化或管理日历首选项设置时。

## 核心原则与规范
- **节假日感知**: 使用 `cc` 子命令列出并选择国家节假日代码。
- **持久化配置**: 使用 `cfg` 在不同会话之间维护国家代码等设置。

## 实战示例

### 打印本月
```bash
# 显示当前月份的日历
x gcal
```

### 年历
```bash
# 显示 2025 年的全年日历
x gcal 2025
```

### 配置节假日
```bash
# 设置 gcal 以使用中国法定节假日
x gcal --cfg cc=CN
```
