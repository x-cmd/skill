# ccal - 中国农历

`ccal` 模块是专为对中国农历感兴趣的用户设计的终端日历。它提供农历日期、节气以及关于传统节日和每日黄历的信息。

## 激活时机
- 当用户想要查看当前或特定日期的农历信息时。
- 当检查中国的节气、公众假期或调休上班日期时。
- 当交互式地查看传统节日日期或生肖信息时。

## 核心原则与规范
- **交互式浏览**: 使用 `fz` 子命令获得可搜索的 TUI 体验。
- **详细信息**: 使用 `info` 子命令获取单日的全面分解（农历、干支等）。
- **本地化**: 支持自定义星期命名（例如使用“礼拜”与“星期”）。

## 实战示例

### 今日农历信息
```bash
# 查看今天的详细农历和传统信息
x ccal info
```

### 特定日期
```bash
# 检查未来日期的农历信息
x ccal info 2025-5-20
```

### 交互式年视图
```bash
# 交互式浏览全年的传统节日和生肖
x ccal fzy
```
