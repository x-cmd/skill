# facl - 文件访问控制列表管理

`facl` 模块简化了与文件访问控制列表 (ACL) 的交互，提供了一种更简单的方法来查看和设置超出标准 Unix rwx 位之外的复杂文件权限。

## 激活时机
- 当用户需要审核文件或目录上的细粒度权限时。
- 当使用 ACL 设置特定的用户或组权限时。

## 核心原则与规范
- **直观显示**: 旨在以清晰、易读的格式显示 ACL 条目。
- **标准操作**: 支持 `get`（查看）和 `set`（修改）子命令。

## 实战示例

### 查看 ACL
```bash
# 获取当前目录的 ACL
x facl
# 获取特定文件的 ACL
x facl myfile.txt
```
