# last - 增强型登录历史查看

`last` 模块提供了一种高度灵活的方式来查看系统登录历史，支持交互式探索、按重启进行树状分组以及结构化数据导出 (JSON/CSV)。

## 激活时机
- 当用户想要审核最近的系统登录和注销活动时。
- 当以结构化格式检查系统重启历史时。
- 当在 TTY 中对用户会话进行交互式分析时。
- 当为安全报告或外部分析导出登录记录时。

## 核心原则与规范
- **交互式浏览**: TTY 中的默认行为是记录的交互式选择器。
- **树状视图**: 使用 `--tree` 按会话之前的重启事件进行分组。
- **结构化格式**: 支持 `-j` (JSON) 和 `-c` (CSV) 以便于解析。

## 实战示例

### 交互式审核
```bash
# 交互式浏览所有登录/注销记录
x last
```

### 重启历史
```bash
# 仅显示系统重启记录
x last --reboot
```

### JSON 导出
```bash
# 以 JSON 对象形式获取所有登录历史
x last --json
```

### 分组视图
```bash
# 以树状结构显示按系统重启分组的登录记录
x last --tree
```
