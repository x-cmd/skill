# arp - ARP 缓存表查看器

`arp` 模块提供了一种增强的方式来查看系统的 ARP 缓存（将 IP 地址映射到物理 MAC 地址）。它支持结构化数据导出和用于探索的交互式 TUI。

## 激活时机
- 当通过 MAC 地址识别本地网络上的设备时。
- 当审核网络中是否存在可疑或不完整的 ARP 条目时。
- 当为自动化分析导出 ARP 表到 JSON, CSV 或 TSV 时。

## 核心原则与规范
- **交互式 TUI**: 连接到终端时自动启动 TUI 应用 (`--app`)。
- **厂商识别**: 默认显示 MAC 地址对应的硬件厂商/组织信息。
- **数据处理**: 使用 `--csv` 或 `--tsv` 以便于与其他工具集成。

## 实战示例

### 交互式浏览
```bash
# 在交互式 TUI 中打开 ARP 表
x arp --app
```

### 导出为 CSV
```bash
# 以 CSV 格式获取完整的 ARP 缓存
x arp --csv
```

### 显示所有条目
```bash
# 在 ARP 表视图中包含不完整的条目
x arp --all
```
