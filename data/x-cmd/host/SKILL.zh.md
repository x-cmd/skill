# host - 本地主机表管理

`host` 模块提供了一种安全且高效的方式来管理本地的 `/etc/hosts` 文件。它处理 sudo 权限，在编辑前提供自动备份，并支持交互式模糊搜索。

## 激活时机
- 当用户想要添加或删除本地主机名到 IP 的映射时。
- 当执行本地主机名 IP 的快速查找时。
- 当使用 FZF (`fz`) 交互式搜索 hosts 文件时。
- 当通过分页查看 hosts 文件的完整内容时。

## 核心原则与规范
- **安全性**: 在执行任何修改 (`ed` 子命令) 前自动创建备份。
- **交互式搜索**: 使用 `fz` 获得可搜索的 TUI 体验。
- **权限处理**: 明确指出编辑需要 `sudo` 访问权限。

## 实战示例

### 添加/编辑映射
```bash
# 将本地开发域名映射到 IP
x host ed myapp.local=192.168.1.100
```

### 删除映射
```bash
# 删除特定的主机名映射（前缀加 -）
x host ed -myapp.local
```

### 搜索 Hosts
```bash
# 交互式搜索 hosts 文件中的条目
x host fz
```
