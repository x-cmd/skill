# colr - 终端颜色查找器

`colr` 模块帮助用户查找 ANSI 颜色编码（8色, 256色）和 RGB 等效项。它还提供了一个用于彩色文本输出的彩虹生成器。

## 激活时机
- 当查找特定颜色（如绿色）的数字编码时。
- 当为特定 RGB 或十六进制值查找近似颜色时。
- 当为消息生成“彩虹”样式的输出时。

## 实战示例

### 查找颜色编码
```bash
# 列出所有 256 色编码及其视觉近似值
x colr
```

### 按十六进制搜索
```bash
# 查找与十六进制值最接近的终端颜色
x colr #008000
```

### 彩虹输出
```bash
# 以彩虹色输出消息
x cowsay "你好" | x colr rainbow
```
