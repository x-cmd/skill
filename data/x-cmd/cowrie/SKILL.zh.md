# cowrie - 基于 Docker 的蜜罐部署

`cowrie` 模块简化了 Cowrie 蜜罐的部署，这是一款旨在模仿 SSH 和 Telnet 服务以记录攻击者行为并收集恶意软件的交互式工具。

## 激活时机
- 当用户想要为安全研究设置蜜罐环境时。
- 当监控未经授权的登录尝试并捕获攻击者命令时。
- 当管理 Cowrie Docker 容器的生命周期（运行、停止、重启）时。

## 核心原则与规范
- **Docker 依赖**: 需要在宿主系统上安装并运行 Docker。
- **隔离测试**: 使用 `test` 子命令在内部验证蜜罐连通性。
- **日志记录**: 优先使用 `log` 子命令来分析捕获的数据。

## 实战示例

### 运行蜜罐
```bash
# 启动默认的 Cowrie 蜜罐容器
x cowrie run
```

### 测试连通性
```bash
# 本地测试到正在运行的蜜罐的 SSH 连接（端口 2223）
x cowrie test
```

### 查看日志
```bash
# 查看来自 Cowrie 蜜罐的实时日志
x cowrie log
```
