# grep - 交互式模式搜索

`grep` 模块通过集成 FZF 增强了标准的全局正则表达式搜索工具，提供了一个交互式环境来搜索文件并实时查看结果。

## 激活时机
- 当用户想要在文件或目录中搜索文本模式时。
- 当需要交互式的、可搜索的 grep 匹配列表时 (`--fz`)。
- 当通过终端执行快速的代码或日志调查时。

## 核心原则与规范
- **交互优先**: 使用 `--fzfapp`（或默认的 `x grep`）获得动态搜索体验。
- **脚本回退**: 通过 `cmd` 子命令支持非交互式使用的标准 `grep` 标志。
- **上下文支持**: 全面支持 `-A`, `-B` 和 `-C` 等标准标志以显示上下文行。

## 实战示例

### 交互式搜索
```bash
# 在当前目录启动交互式搜索
x grep
```

### 通过管道搜索
```bash
# 将命令输出通过管道传输到交互式 grep 会话中
x ascii | x grep
```

### 搜索特定目录
```bash
# 打开交互式 TUI 在特定路径内搜索
x grep --fzfapp ~/.x-cmd.root
```
