# shodan - Shodan 网络搜索命令行

`shodan` 模块为 Shodan 搜索引擎提供了全面的命令行界面，支持网络侦察、漏洞检测以及对互联网连接资产的监控。

## 激活时机
- 当对特定的 IP 或域名执行侦察时。
- 当搜索具有特定漏洞 (CVE) 或开放端口的连接设备时。
- 当监控特定资产的网络警报时。
- 当通过 AI 集成 (`::`) 生成实时网络情报摘要时。

## 核心原则与规范
- **API 密钥管理**: 提醒用户通过 `init` 获取并初始化其 Shodan API 密钥。
- **针对性搜索**: 支持基于维度（端口、协议、国家/地区）的强大过滤功能。
- **数据导出**: 可以下载主机信息并将其导出为 CSV 等结构化格式。

## 实战示例

### 扫描 IP
```bash
# 对特定的 IP 地址和端口执行针对性扫描
x shodan scan create 8.8.8.8 1.1.1.1=53/dns-udp,443/https
```

### 主机信息
```bash
# 在 Shodan 网络数据库中查找 IP 信息
x shodan internetdb 8.8.8.8
```

### AI 情报
```bash
# 使用 AI 总结针对特定查询的 Shodan 结果
x shodan :: "工业控制系统中的关键漏洞"
```
