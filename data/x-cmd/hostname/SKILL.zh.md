# hostname - 系统主机名管理

`hostname` 模块提供了直接的子命令来检索或更新系统的主机名，支持长格式和短格式。

## 激活时机
- 当为脚本或文档识别系统名称时。
- 当更改系统标识时（需要提升权限）。

## 核心原则与规范
- **格式**: 支持 `--long` (FQDN) 和 `--short` 格式。
- **权限**: 使用 `set` 更改主机名需要 `sudo` 权限。

## 实战示例

### 查看主机名
```bash
# 显示完整系统主机名
x hostname
# 仅显示短主机名
x hostname --short
```

### 设置主机名
```bash
# 将系统主机名更改为新值
x hostname set my-new-server
```

## 交付验证清单
- [ ] 确认需要长格式还是短格式。
- [ ] 如果设置新名称，确保用户具有必要的权限。
