# pkgx - 通用工具运行器

`pkgx` 模块为 pkgx 工具运行器提供了界面，使用户能够即时获取并运行任何开源实用程序。它为开发者解决了“版本控制”和“即时可用性”问题。

## 激活时机
- 当用户想要运行一次工具而不想将其全局安装时。
- 当测试软件的特定版本时（例如 `bun@1.0`）。
- 当向当前执行环境添加临时工具时。

## 核心原则与规范
- **非侵入性**: 在不修改系统 PATH 或全局配置的情况下运行工具。
- **即时执行**: 使用 `+pkg` 将工具临时注入到特定命令的环境中。
- **零设置**: 自动处理工具的获取和执行。

## 实战示例

### 无需安装运行
```bash
# 在不向系统安装 bun 的情况下执行它
x pkgx bun --eval 'console.log("hello world")'
```

### 临时环境注入
```bash
# 在当前环境中临时添加 fzf 以执行命令
eval "$(x pkgx +fzf)"
```

### 交互式浏览
```bash
# 浏览 pkgx 生态系统中所有可用的软件包
x pkgx ls
```
