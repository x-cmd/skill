# hash - 加密校验和工具

`hash` 模块为计算 MD5、SHA1、SHA256、SHA384 和 SHA512 哈希提供了一个统一的界面。它包括用于断言匹配以及通过 cosmo 实用程序加速计算的工具。

## 激活时机
- 当验证下载文件（如 ISO, 软件包）的完整性时。
- 当为要共享或归档的文件生成校验和时。
- 当在脚本中执行静默哈希比较时 (`match`)。
- 当同时检查多个文件时。

## 核心原则与规范
- **多算法**: 在验证时根据哈希长度自动检测算法。
- **加速**: 支持 `speedup` 子命令以安装优化的哈希工具。
- **脚本就绪**: 在自动化中使用 `match` 进行静默的成功/失败检测。

## 实战示例

### 计算哈希
```bash
# 获取文件的 SHA256 校验和
x hash sha256 myfile.txt
```

### 断言匹配
```bash
# 根据预期哈希验证文件，并提供视觉反馈
x hash assert myfile.txt 预期哈希字符串
```

### 批量处理
```bash
# 计算目录下所有 .txt 文件的 MD5
x hash md5 *.txt
```
