# stat - 增强型文件与文件系统状态查看

`stat` 模块提供文件和文件系统的详细元数据，包括权限、所有权、大小和时间戳，并能够以结构化格式导出这些数据。

## 激活时机
- 当用户需要详细的文件属性（权限、inode、上次修改时间）时。
- 当检查文件系统使用情况和限制（例如总空间、空闲 inode）时。
- 当为自动化分析导出文件元数据到 JSON, CSV 或 TSV 时。
- 当使用交互式 TUI 浏览文件详情时。

## 核心原则与规范
- **精确性**: 提供超出标准 `ls` 输出的详细、技术性元数据。
- **导出格式**: 为数据处理任务优先使用 `--json` 或 `--csv`。
- **文件系统支持**: 使用 `-f` 标志针对整个文件系统而不是单个文件。

## 实战示例

### 文件元数据
```bash
# 查看特定文件的详细信息
x stat myfile.txt
```

### 文件系统状态
```bash
# 检查根文件系统的状态
x stat -f /
```

### JSON 导出
```bash
# 以 JSON 对象形式获取文件状态
x stat --json myfile.txt
```
