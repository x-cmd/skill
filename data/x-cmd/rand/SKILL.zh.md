# rand - 随机数据生成

`rand` 模块生成各种类型的随机数据，从基础数字和字符串到 UUIDv7 等复杂标识符，以及邮箱或 IP 地址等模拟数据。

## 激活时机
- 当生成唯一标识符（UUID, UUIDv7）时。
- 当创建随机密码或令牌（字符串、字母数字）时。
- 当为测试生成模拟数据（邮箱、IP）时。
- 当需要特定范围内的随机整数或浮点数时。

## 核心原则与规范
- **范围控制**: 为 `int` 和 `float` 使用特定的最小值/最大值以适应任务。
- **批量生成**: 支持同时生成多个项目（例如 `x rand email 5`）。
- **标识符唯一性**: 优先使用 `uuidv7` 以获得按时间排序的唯一标识符。

## 实战示例

### 生成标识符
```bash
# 创建标准 UUID 和 UUIDv7
x rand uuid
x rand uuidv7
```

### 随机字符串
```bash
# 生成 16 位随机字母数字字符串
x rand alphanum 16
```

### 模拟测试数据
```bash
# 生成 10 个随机 IPv4 地址
x rand ip 10
```
