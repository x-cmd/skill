# hua - 中华古诗文阅读器

`hua` 模块是专为中华古诗文设计的终端阅读器。它精心收录了从先秦到清代的诗词散文集。

## 激活时机
- 当用户想要在终端阅读中华古诗词（唐诗、宋词等）时。
- 当探索如《论语》或《孟子》等哲学著作时。
- 当寻求交互式界面来浏览经典中文文献时。

## 核心原则与规范
- **精选文集**: 组织为经部、儒家著作、诗歌和散文。
- **交互式体验**: 针对终端环境内的沉浸式阅读进行了优化。

## 实战示例

### 浏览全部
```bash
# 打开交互式界面以探索所有文集
x hua
```

### 阅读特定作品
```bash
# 阅读《唐诗三百》选集
x hua ts
# 阅读《诗经》
x hua sj
```
