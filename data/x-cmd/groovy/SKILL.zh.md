# groovy - Groovy 语言管理

`groovy` 模块为 Groovy 编程语言提供了界面，促进了脚本执行以及与 Java 生态系统的集成。

## 激活时机
- 当运行 Groovy 脚本 (`.groovy`) 时。
- 当管理 Groovy 版本和依赖项时。

## 实战示例

### 运行 Groovy 脚本
```bash
# 执行一个 Groovy 文件
x groovy ./myscript.groovy
```
