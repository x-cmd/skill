# kev - 已知被利用漏洞清单

`kev` 模块提供了一个简单的界面来列出和查询已知被利用的漏洞，帮助安全专业人员根据现实世界的利用数据优先处理补丁。

## 激活时机
- 当对当前安全威胁执行快速审计时。
- 当列出已知在野外活跃的顶级或最新漏洞时。

## 核心原则与规范
- **直接检索**: 从权威的安全目录中获取数据。
- **过滤**: 支持查看完整列表或仅查看前 N 个最关键/最新的条目。

## 实战示例

### 列出所有 KEV
```bash
# 显示完整的已知被利用漏洞目录
x kev ls
```

### 顶级关键 KEV
```bash
# 查看 KEV 清单中的前 100 条记录
x kev top 100
```
