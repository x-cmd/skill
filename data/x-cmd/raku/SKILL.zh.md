# raku - Raku 语言管理

`raku` 模块为 Raku 编程语言提供了界面，简化了脚本执行和环境设置。

## 激活时机
- 当运行 Raku 脚本或与 Raku 生态系统交互时。

## 实战示例

### 运行 Raku 脚本
```bash
# 执行一个 Raku 文件
x raku ./myscript.raku
```
