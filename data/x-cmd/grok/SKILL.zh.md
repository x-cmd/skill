# grok - xAI Grok 集成

`grok` 模块为 xAI 的 Grok 模型提供命令行界面，支持强大的文本生成和推理。

## 激活时机
- 当用户想要与 Grok 聊天时。
- 当执行实时查询或复杂的推理任务时。
- 当管理 xAI API 密钥和模型配置时。

## 核心原则与规范
- **API 密钥管理**: 使用 `init` 或 `--cfg apikey=<key>` 进行设置。
- **实验性状态**: 请注意，此模块目前处于公开实验阶段。
- **别名访问**: 使用 `@grok` 别名进行快速交互。

## 实战示例

### 与 Grok 聊天
```bash
# 通过终端向 Grok 提问
@grok "太空探索的最新突破是什么？"
```
