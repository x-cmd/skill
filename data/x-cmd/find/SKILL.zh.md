# find - 增强型文件查找

`find` 模块扩展了标准的系统 find 实用程序，提供了一个由 FZF 驱动的交互式应用，用于快速发现文件和基于元数据的过滤。

## 激活时机
- 当用户想要根据模式或属性定位文件或目录时。
- 当需要交互式的、可搜索的文件列表时 (`--fzfapp`)。
- 当按大小（如大于 100MB）或修改时间查找文件时。

## 核心原则与规范
- **交互模式**: 如果未给出特定标准，会自动启动 TUI 应用以便于浏览。
- **基于属性**: 支持标准的 find 标志，如 `-type`, `-name`, `-size` 和 `-mtime`。
- **可操作结果**: 可以使用 `-exec` 对找到的项目执行命令。

## 实战示例

### 交互式发现
```bash
# 打开交互式 FZF 应用以查找文件或文件夹
x find
```

### 按名称搜索
```bash
# 查找当前目录下的所有文本文件
x find . -name "*.txt"
```

### 大文件搜索
```bash
# 在家目录中查找大于 100MB 的文件
x find /home -size +100M
```
