# jq - JSON 处理器与交互式 REPL

`jq` 模块为处理 JSON 数据提供了强大的命令行界面。x-cmd 版本通过提供由 FZF 驱动的交互式 REPL，增强了探索复杂数据结构的体验。

## 激活时机
- 当用户需要提取特定字段或转换 JSON 结构时。
- 当交互式地探索大型 JSON 文件或 API 响应时。
- 当过滤数组或根据特定标准选择对象时。
- 当美化打印 JSON 以获得更好的可读性时。

## 核心原则与规范
- **交互式探索**: 使用 `repl` (或 `r`) 子命令进行带有实时反馈的数据探索。
- **管道支持**: 针对从其他命令（如 `curl ... | x jq`）接收数据进行了高度优化。
- **零依赖**: 如果环境中缺少 `jq`，会自动处理其安装。

## 实战示例

### 提取字段
```bash
# 从 JSON 文件中获取 'name' 字段
x jq '.name' data.json
```

### 交互式 REPL
```bash
# 交互式地探索 API 响应
curl https://api.example.com/data | x jq r
```

### 过滤数组
```bash
# 选择所有年龄大于 18 岁的用户
x jq '.[] | select(.age > 18)' users.json
```
