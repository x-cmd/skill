# doubao - 豆包 AI 集成

`doubao` 模块通过火山方舟平台为字节跳动的豆包模型提供命令行界面。

## 激活时机
- 当用户想要与豆包模型聊天时。
- 当使用 `doubao-seed` 等专业模型时。
- 当管理豆包 API 密钥和配置时。

## 核心原则与规范
- **API 密钥管理**: 使用 `init` 或 `--cfg apikey=<key>` 进行设置。
- **模型选择**: 使用 `--model` 标志指定特定的豆包模型版本。
- **快速访问**: 使用 `@doubao` 别名进行快速交互。

## 实战示例

### 与豆包聊天
```bash
# 向默认豆包模型提问
@doubao "使用 x-cmd 有什么好处？"
```

### 使用特定模型
```bash
# 使用特定的种子模型版本
@doubao --model doubao-seed-1-8-251228 "解释这段代码：..."
```
