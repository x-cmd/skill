# tping - TCP 端口连通性测试

`tping` 模块使用 `curl` 对目标主机的特定端口执行 TCP 连接测试。当 ICMP（标准 ping）被阻止时，它是验证服务可用性的理想选择。

## 激活时机
- 当测试 Web 服务器（80/443 端口）或自定义服务是否可达时。
- 当目标的防火墙禁用了标准 ICMP ping 时。
- 当需要 TCP 连接时间的视觉指标（热图/条形图）时。

## 核心原则与规范
- **端口针对性**: 允许测试到任何 TCP 端口的连通性（默认为 80）。
- **可视化**: 支持与增强型 `ping` 模块相同的热图和条形图模式。

## 实战示例

### 基础端口检查
```bash
# 测试到域名 80 端口的 TCP 连通性
x tping bing.com
```

### 带可视化的自定义端口
```bash
# 以实时条形图检查 443 端口
x tping --bar bing.com:443
```
