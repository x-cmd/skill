# ping - 增强型网络连通性测试

`ping` 模块为标准的连通性测试提供了增强型界面，支持多种可视化模式，以更好地了解网络性能和延迟。

## 激活时机
- 当检查远程主机是否通过 ICMP 可达时。
- 当需要网络延迟随时间变化的视觉表示（热图、条形图）时。
- 当为分析导出连通性数据到 CSV 或 TSV 时。

## 核心原则与规范
- **可视化**: 使用 `--heatmap` 或 `--bar` 获取关于延迟稳定性的即时视觉反馈。
- **数据友好**: 支持结构化输出格式，以便与监控工具集成。

## 实战示例

### 视觉监控
```bash
# Ping 主机并显示延迟的实时热图
x ping --heatmap bing.com
```

### 数据导出
```bash
# Ping 主机 10 次并以 CSV 格式输出结果
x ping -c 10 --csv 8.8.8.8
```
