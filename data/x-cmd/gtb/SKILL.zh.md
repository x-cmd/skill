# gtb - Project Gutenberg 命令行浏览器

`gtb` 模块为 Project Gutenberg 提供了界面，允许用户搜索其免费电子书库、查看文本内容并从 CLI 访问元数据。

## 激活时机
- 当用户想要搜索经典书籍或作者时（例如莎士比亚、大仲马）。
- 当在终端阅读公有领域书籍的全文时。
- 当通过 CLI 访问书籍元数据或在浏览器中查看书籍时。

## 核心原则与规范
- **文本优先**: 针对检索和查看原始文本内容进行了优化 (`txt` 子命令)。
- **交互式 TUI**: 使用 `show` 获得交互式阅读体验。
- **搜索支持**: 利用 DuckDuckGo 进行与 Gutenberg 内容相关的更广泛查询。

## 实战示例

### 按作者搜索
```bash
# 搜索亚历山大·大仲马的书籍
x gtb search Dumas
```

### 查看书籍文本
```bash
# 检索书籍 ID 100 的全文
x gtb txt 100
```

### 交互式阅读
```bash
# 在交互式界面中浏览书籍 ID 100
x gtb show 100
```

## 交付验证清单
- [ ] 确认用户是拥有特定的书籍 ID 还是需要执行搜索。
- [ ] 验证用户是想要在终端阅读还是在浏览器中打开。
