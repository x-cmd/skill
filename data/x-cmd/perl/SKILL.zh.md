# perl - Perl 语言管理

`perl` 模块为在各种平台上执行 Perl 脚本和管理 Perl 环境提供了一个统一的界面。

## 激活时机
- 当在终端中运行 Perl 脚本时。
- 当识别或管理 Perl 运行时。

## 实战示例

### 运行脚本
```bash
# 执行本地 Perl 脚本
x perl ./script.pl
```
