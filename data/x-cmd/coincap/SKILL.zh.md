# coincap - 加密货币市场数据

`coincap` 模块将 CoinCap API 集成到终端中，为 1,000 多种加密货币提供实时数据。它支持交互式浏览以及 JSON 或 CSV 格式的数据导出。

## 激活时机
- 当用户想要检查当前的加密货币价格和市场统计数据时。
- 当使用结构化加密数据执行自动化财务分析时。
- 当终端中需要加密市场的交互式概览时。

## 核心原则与规范
- **需要 API 密钥**: 提醒用户获取 API 密钥以进行高频率使用。
- **数据便携性**: 为基于脚本的市场分析优先使用 `--json` 和 `--csv`。
- **交互式 TUI**: 使用 `--app` 获得视觉化的表格浏览体验。

## 实战示例

### 查看市场 (交互式)
```bash
# 为实时加密货币价格打开交互式 TUI
x coincap --app
```

### 导出市场数据
```bash
# 以 JSON 对象形式获取所有市场数据
x coincap --json
```

### 初始化配置
```bash
# 设置 API 密钥和会话默认值
x coincap init
```
