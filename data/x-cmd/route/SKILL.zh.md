# route - 网络路由表查看器

`route` 模块提供了一个查看系统网络路由表的界面，允许用户检查网络流量的导向。它支持以结构化格式导出路由表。

## 激活时机
- 当诊断网络连通性或路由问题时。
- 当需要当前路由表的结构化列表 (CSV) 进行分析时。

## 核心原则与规范
- **实验性状态**: 请注意，此模块目前处于实验阶段。
- **数据导出**: 支持 `--csv` 以便于解析。

## 实战示例

### 查看路由表
```bash
# 显示当前路由表
x route
```

### 导出为 CSV
```bash
# 将路由表输出为 CSV
x route --csv
```
