# sleep - 睡眠与周期性任务安排

`sleep` 模块增强了标准 sleep 命令，支持人性化的时间格式，并为重复任务提供了集成的调度功能。

## 激活时机
- 当脚本需要暂停一段时间（如 `5m` 或 `1h30m`）时。
- 当创建一个定期输出时间戳的“信标”(beacon) 时。
- 当安排命令定期运行（轮询、备份、测试）时。

## 核心原则与规范
- **时间单位**: 使用 `s`、`m`、`h`、`d` 直观地指定时长。
- **组合命令**: 使用 `x sleep <时间> <命令>` 暂停后立即执行命令。
- **调度循环**: 使用 `schd` 进行受控的重复任务，支持可选的迭代限制 (`-n`)。

## 实战示例

### 延迟脚本
```bash
# 暂停 1 分钟后继续
x sleep 1m
```

### 定期时间戳 (Beacon)
```bash
# 每 5 秒输出一次当前时间戳
x sleep beacon 5s
```

### 重复任务
```bash
# 每小时无限循环运行一次备份脚本
x sleep schd -i 1h -- ./backup.sh
```

### 有限重复任务
```bash
# 每 10 秒运行一次测试命令，共执行 5 次
x sleep schd -i 10s -n 5 -- make test
```
