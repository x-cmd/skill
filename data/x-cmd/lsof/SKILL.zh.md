# lsof - 列出打开文件增强工具

`lsof` 模块提供了一种更直观的方式来与系统的打开文件列表进行交互。它支持用于探索的交互式 TUI，并允许以常见数据格式导出结果。

## 激活时机
- 当调查文件锁定问题或识别文件的进程所有权时。
- 当使用 FZF 对打开的文件执行交互式搜索时。
- 当为脚本处理导出打开的文件数据到 CSV 或 TSV 时。
- 当需要打开文件的线程级详情 (`--task`) 时。

## 核心原则与规范
- **交互式 TUI**: 连接到终端时自动显示 TUI；否则，默认为 TSV。
- **以数据为中心**: 当输出打算用于自动化或外部分析时，使用 `--csv` 或 `--tsv`。
- **可搜索性**: 利用 `fz` 子命令进行快速的交互式过滤。

## 实战示例

### 交互式搜索
```bash
# 通过 FZF 交互式搜索并查看打开的文件
x lsof fz
```

### 导出为 CSV
```bash
# 将当前打开的文件输出为 CSV 以供处理
x lsof --csv
```

### 查看线程信息
```bash
# 显示打开文件的任务 (线程) ID 和名称
x lsof --task
```
