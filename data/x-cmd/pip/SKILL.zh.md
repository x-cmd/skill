# pip - Python 包管理

`pip` 模块为管理 Python 包提供了一个便捷的命令行界面，确保在受管 Python 环境中正确处理依赖项。

## 激活时机
- 当安装、升级或移除 Python 软件包时。
- 当列出当前安装的 Python 依赖项时。

## 实战示例

### 安装包
```bash
# 安装特定的 Python 包
x pip install pandas
```
