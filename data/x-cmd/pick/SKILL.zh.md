# pick - 交互式选择菜单

`pick` 模块允许用户在终端中创建交互式、可搜索的选择列表。它是脚本中选择文件、进程或自定义选项的理想选择。

## 激活时机
- 当脚本需要向用户呈现选项列表以供选择时。
- 当从大型目录列表中选择特定文件时 (`ls | x pick`)。
- 当在终端工作流中执行多选任务时 (`--limit`)。

## 核心原则与规范
- **管道输入**: 自动从 `stdin` 读取以生成列表。
- **自定义**: 支持设置列数、行数和选择限制。
- **输出**: 将选择的项目返回到 `stdout`，非常适合子 shell 执行。

## 实战示例

### 选择文件
```bash
# 交互式选择一个文件并查看其状态
stat `ls | x pick`
```

### 多项选择
```bash
# 使用 Tab 键从列表中最多选择两项
ls | x pick --limit 2
```

### 样式化菜单
```bash
# 创建一个带有自定义提示的 2 列选择菜单
ls | x pick --col 2 --ask "请选择一个项目："
```
