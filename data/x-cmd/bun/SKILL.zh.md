# bun - 高性能 JS/TS 工具集

`bun` 模块为 Bun 提供了一个简化的界面，Bun 是一款专注于速度的全能 JavaScript 运行时、打包器和测试运行器。

## 激活时机
- 当需要高性能的 JavaScript 或 TypeScript 运行时。
- 当使用 Bun 作为 npm 或 yarn 的快速替代方案进行包管理时。
- 当执行能从 Bun 的性能中获益的本地开发任务时。

## 实战示例

### 使用 Bun 运行
```bash
# 使用 Bun 运行时执行脚本
x bun run main.ts
```

### 快速安装
```bash
# 使用 Bun 快速安装项目依赖项
x bun install
```
