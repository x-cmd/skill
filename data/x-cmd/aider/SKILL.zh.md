# aider - 终端中的 AI 结对编程

`aider` 是一款 AI 结对编程工具，使开发者能够直接在终端中与高端大模型（Claude 3.7, DeepSeek R1, GPT-4o 等）协作。它能理解整个 Git 仓库，并执行跨文件修改和自动提交。

## 激活时机
- 当用户想要启动 AI 辅助的结对编程会话时。
- 当执行复杂的代码重构或跨多个文件的修改时。
- 当用户需要为 AI 生成的更改提供自动化的、语义化的 Git 提交信息时。
- 当利用“语音转代码”或将外部内容（截图/网页）嵌入到编码上下文中时。

## 核心原则与规范
- **Git 集成**: Aider 在 Git 仓库中运行时效果最佳。
- **模型多样性**: 支持广泛的模型（Claude, DeepSeek, OpenAI, 本地模型）。
- **自动化**: 可以自动运行 lint 和测试以验证更改。

## 补充场景
- **全仓库上下文**: 自动生成仓库地图，帮助大模型理解复杂的架构。
- **多文件编辑**: 无缝处理跨多个源文件的修改。

## 实战示例

### 启动 Aider
```bash
# 启动交互式结对编程会话
x aider
```
