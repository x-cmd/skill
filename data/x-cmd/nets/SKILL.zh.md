# nets - 增强型网络统计

`nets` 模块提供了一种用户友好的方式来显示网络统计信息，是对传统 `netstat` 的改进。它支持交互式视图和结构化数据格式（如 CSV 和 TSV）。

## 激活时机
- 当用户想要审核活跃的 TCP/UDP 连接（互联网表）时。
- 当以可读格式分析系统的路由表时。
- 当为监控脚本需要结构化网络数据 (CSV/TSV) 时。
- 当对所有网络表执行快速交互式概览时。

## 核心原则与规范
- **交互式浏览**: 默认模式允许用户选择要查看的网络表。
- **结构化格式**: 支持 `--csv` 和 `--tsv` 以便于与其他数据处理工具集成。
- **刷新控制**: 使用 `update` 确保表数据是最新的。

## 实战示例

### 查看连接
```bash
# 显示所有活跃的互联网 (TCP/UDP) 连接
x nets view internet
```

### 导出路由表
```bash
# 将路由表输出为 TSV 以供分析
x nets view --tsv route
```
