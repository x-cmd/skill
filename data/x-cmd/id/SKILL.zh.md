# id - 增强型用户身份查看

`id` 模块为标准 `id` 命令提供了一个更具可读性且彩色的替代方案，以结构化的键值格式显示用户和组 ID 及其成员关系。

## 激活时机
- 当用户想要查看自己或其他用户的身份信息 (UID, GID) 时。
- 当脚本需要仅检索数字 UID 或 GID 时。
- 当手动验证组关系时。

## 核心原则与规范
- **可读性**: 在 TTY 会话中自动提供彩色输出。
- **脚本友好**: 使用 `uid_` 或 `gid_` 子命令将结果直接存储在变量中以进行自动化。
- **原生回退**: 对于未知选项，回退到标准行为。

## 实战示例

### 查看身份
```bash
# 为当前用户显示彩色的身份信息
x id
```

### 检查特定用户
```bash
# 查看 'root' 的身份详情
x id root
```

### 获取数字 UID
```bash
# 仅输出数字形式的用户 ID
x id uid
```
