# osv - 开源漏洞扫描器

`osv` 模块为 OSV 项目提供了界面，使用户能够扫描本地项目、识别有漏洞的依赖项并获取详细的漏洞信息。

## 激活时机
- 当用户想要对其项目依赖项（npm, pip 等）执行安全审计时。
- 当查询特定漏洞 ID（如 `osv-2020-111`）的详细信息时。
- 当生成 SARIF 格式的安全报告以便与 CI/CD 流水线集成时。
- 当搜索与特定软件包和版本相关的漏洞时。

## 核心原则与规范
- **全面扫描**: 使用 `sarif` 生成标准化的安全报告。
- **生态系统感知**: 支持包括 npm、pypi 在内的多个生态系统。
- **搜索集成**: 利用 AI 或 DuckDuckGo 从网络总结漏洞详情。

## 实战示例

### 全项目扫描
```bash
# 扫描依赖并生成 SARIF 报告
x osv sarif
```

### 特定漏洞查询
```bash
# 获取特定漏洞 ID 的详情
x osv vuln OSV-2020-111
```

### 检查软件版本
```bash
# 查询特定版本软件包的漏洞
x osv q -p jq -v 1.7.1
```
