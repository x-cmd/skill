# ps - 增强型进程状态查看

`ps` 模块通过添加交互式 TUI 功能以及将进程快照导出为 JSON 和 CSV 等结构化数据格式的能力，扩展了标准的进程查看功能。

## 激活时机
- 当用户想要使用交互式 UI（FZF 或 CSV app）浏览进程时。
- 当为脚本编写将当前进程列表导出为 JSON、CSV 或 TSV 时。
- 当将传统的 `ps aux` 输出转换为结构化数据格式时。

## 核心原则与规范
- **交互式探索**: 使用 `fz` 获得快速、可搜索的 TUI 体验。
- **数据导出**: 当用户打算使用其他工具（如 `jq`）处理数据时，优先使用 `--json` 或 `--csv`。
- **管道兼容性**: 可以接收标准 `ps` 命令的输出并对其进行转换。

## 实战示例

### 交互式 UI
```bash
# 在可搜索的交互式 TUI 中查看进程
x ps fz
```

### 导出为 JSON
```bash
# 将当前进程信息输出为 JSON 数组
x ps --json
```

### 数据转换
```bash
# 将特定的 ps 命令输出转换为 CSV
ps -ef | x ps --tocsv
```
