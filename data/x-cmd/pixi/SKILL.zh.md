# pixi - 快速包与环境管理器

`pixi` 模块为 Pixi 提供了增强型界面，提供镜像加速下载，并使用 x-cmd 惯例简化了全局包管理。

## 激活时机
- 当用户想要在其用户环境中全局安装工具时 (`use`)。
- 当管理具有隔离依赖项的基于 Conda 的项目时。
- 当创建一个新项目或向现有 pixi 项目添加依赖项时。
- 当在临时的、隔离的环境中运行命令时 (`exec`)。

## 核心原则与规范
- **全局管理**: 对于全系统工具安装，优先使用 `x pixi use <pkg>`。
- **镜像加速**: 自动使用优化的镜像以在特定地区加速下载。
- **环境隔离**: 支持使用 `shell` 和 `exec` 在项目的锁定环境中运行代码。

## 实战示例

### 安装全局工具
```bash
# 通过 pixi 全局安装 'fish' shell 和 'jq'
x pixi use fish jq
```

### 隔离的命令执行
```bash
# 在临时的 pixi 环境中运行命令
x pixi exec python --version
```

### 交互式浏览
```bash
# 交互式搜索可用的 pixi 软件包
x pixi la
```
