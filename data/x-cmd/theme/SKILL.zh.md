# theme - 终端 UI 与主题管理

`theme` 模块提供了一套全面管理终端命令行及 x-cmd 交互组件视觉风格的方法。它支持实时预览、永久设置以及基于会话的尝试。

## 激活时机
- 当用户想要浏览并选择新的终端主题时。
- 当更改 x-cmd 组件的全局显示风格时。
- 当在不进行永久更改的情况下为当前会话尝试主题时。
- 当管理 Claude Code 的状态栏组件时。

## 核心原则与规范
- **交互式选择**: 使用 `--app` 模式获得视觉化的、交互式的主题选择体验。
- **持久性**: 使用 `use` 进行永久更改，使用 `try` 进行仅限会话的临时更改。
- **去重**: 除非设置了 `___X_CMD_THEME_RELOAD_DISABLE`，否则自动处理主题重新加载。

## 实战示例

### 交互式主题浏览器
```bash
# 打开主题预览应用以选择样式
x theme --app
```

### 应用主题
```bash
# 将 'robby' 永久设置为全局命令行主题
x theme use robby
```

### 尝试主题
```bash
# 仅在当前终端会话中测试主题
x theme try festival
```
