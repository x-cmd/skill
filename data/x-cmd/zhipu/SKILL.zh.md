# zhipu - 智谱 AI (GLM) 集成

`zhipu` 模块提供智谱 AI 的 GLM（通用语言模型）系列命令行界面，支持强大的文本生成和推理。

## 激活时机
- 当用户想要使用智谱 GLM 模型进行聊天或文本分析时。
- 当管理智谱 API 密钥和会话默认值时。
- 当使用 `@glm` 别名进行快速访问时。

## 核心原则与规范
- **API 密钥管理**: 使用 `init` 或 `--cfg apikey=<key>` 进行设置。
- **模型选择**: 使用 `--model` 标志指定模型（例如 `glm-4`）。
- **别名访问**: 优先使用 `@glm` 别名以获得更快的 CLI 体验。

## 实战示例

### 与 GLM 聊天
```bash
# 使用 GLM 模型提问
@glm "长城的历史是什么？"
```

### 使用特定版本
```bash
# 使用特定模型版本进行推理
@glm --model glm-4.7 "strawberry 这个单词中有多少个 R？"
```
