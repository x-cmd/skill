# str - 高性能字符串处理

`str` 模块是一个用于常见字符串操作的底层、高性能工具。它是 Shell 脚本内数据处理的基础，同时支持管道输入和直接参数输入。

## 激活时机
- 当转换字符串大小写时。
- 当修剪空白或按分隔符分割字符串时。
- 当执行 Base64 编码/解码或计算 MD5/SHA256 哈希时。
- 当替换子串或连接多个字符串时。
- 当转换换行符（dos2unix/unix2dos）时。

## 核心原则与规范
- **管道支持**: 针对终端管道进行了高度优化（例如 `echo "..." | x str upper`）。
- **Python 风格切片**: 对于复杂的子串提取，使用 `slice` 子命令。
- **高效性**: 旨在减少高频脚本调用中的开销。

## 补充场景
- **哈希验证**: 快速为字符串或数据流生成哈希。
- **格式转换**: 使用 `dos2unix` 在不同的操作系统环境之间标准化文本文件。

## 实战示例

### 基本转换
```bash
# 将字符串转换为大写
x str upper "hello world"
```

### 分割与连接
```bash
# 按逗号分割并按空格连接
echo "a,b,c" | x str split "," | x str join " "
```

### 编码与哈希
```bash
# Base64 编码并生成 SHA256 哈希
x str base64 "secret"
x str sha256 "data"
```
