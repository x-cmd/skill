# cd - 增强型目录导航

`cd` 模块是标准 `cd` 命令的强大替代方案，使用 Shell 和 awk 实现。它提供交互式历史导航、基于关键词的跳转以及目录别名功能。

## 激活时机
- 当用户想要根据路径中的关键词跳转到目录时。
- 当使用关键词匹配回退到上级目录时 (`-b`)。
- 当对子目录执行前向搜索时 (`-f`)。
- 当管理或使用目录“领域” (别名) 时。
- 当在另一个目录执行命令并立即返回时。

## 核心原则与规范
- **无缝跳转**: 使用 `,` 进行交互式历史搜索，或使用 `,关键词` 进行直接跳转。
- **双向导航**: 使用 `-b` 向“后”跳转（向上级树），使用 `-f` 向“前”跳转（向下级子文件夹）。
- **执行包装器**: 支持在目标目录中执行命令，而不会永久更改会话路径。

## 实战示例

### 智能跳转
```bash
# 打开交互式历史搜索
c
# 跳转到匹配 'project' 的历史路径
c ,project
```

### 上下文导航
```bash
# 向上跳转到最近的匹配 'src' 的上级目录
c -b src
# 查找并跳转到匹配 'test' 的子文件夹
c -f test
```

### 临时执行
```bash
# 在 /bin 中运行 'ls -la' 然后自动返回
c /bin ls -la
```
