# uninstall - 统一的包卸载器

`uninstall` 模块提供了一个统一的界面，用于在包括 Homebrew、Scoop、APT、DNF 等在内的多个包管理器中查询并移除已安装的软件。

## 激活时机
- 当用户想要卸载程序而不必记住是哪个管理器安装的。
- 当对已安装的包执行系统清理时。
- 当卸载 x-cmd 本身时。

## 核心原则与规范
- **交互式清理**: 如果没有给出参数，则启动一个包含所有已安装包的交互式列表。
- **多管理器支持**: 自动检测并委派移除任务给适当的后端（如 brew, scoop）。

## 实战示例

### 交互式卸载
```bash
# 浏览所有已安装的软件并选择要移除的项目
x uninstall
```

### 针对性移除
```bash
# 显式从 Homebrew 卸载 'bat'
x uninstall brew bat
```

### 卸载自身
```bash
# 从系统中移除 x-cmd
x uninstall self
```
