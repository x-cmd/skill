# cal - 增强型格里历日历

`cal` 模块是经典日历命令的增强版本，支持标准的格里历视图以及星座查询等附加功能。

## 激活时机
- 当用户想要查看标准的月历或年历时。
- 当查询特定出生日期的星座时。

## 核心原则与规范
- **本地化**: 自动以首选语言输出星座名称（例如“狮子座”对比“Leo”）。
- **简洁性**: 专注于作为 `cal` 的快速、直接替代品。

## 实战示例

### 查询星座
```bash
# 查找特定日期的星座
x cal zodiac 06-22
```

### 查看未来日历
```bash
# 显示 2024 年 12 月的日历
x cal 12 2024
```
