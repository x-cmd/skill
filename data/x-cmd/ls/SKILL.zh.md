# ls - 增强型资源列表

`ls` 模块扩展了传统的文件列表命令，通过统一的界面提供对各种系统资源（CPU、内存、进程、PATH）的快速访问。

## 激活时机
- 当用户想要在查看文件信息的同时查看系统资源摘要时。
- 当在列表工作流中快速检查 PATH 变量或进程列表时。
- 当使用交互式应用 (`:app`) 浏览文件元数据时。

## 核心原则与规范
- **资源关注**: 使用以 `:` 开头的前缀子命令来针对特定的系统数据。
- **交互式 UI**: 使用 `:app` 获得比标准列表更丰富的元数据探索。

## 实战示例

### 系统摘要
```bash
# 使用增强型 ls 快速查看 CPU 和内存信息
x ls :cpu
x ls :mem
```

### 环境检查
```bash
# 列出 PATH 变量中当前引入的所有目录
x ls :path
```

### 元数据浏览器
```bash
# 打开文件和目录信息的交互式查看器
x ls :app
```
