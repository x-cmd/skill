# man - 增强型手册页查看器

`man` 模块通过添加 FZF 交互式浏览，并集成 ManKier 等 AI 服务来解释复杂的命令行字符串，扩展了标准的系统手册功能。

## 激活时机
- 当用户想要使用关键词搜索手册页时。
- 当需要系统所有手册页的交互式列表时。
- 当用户需要 AI 解释特定命令标志（如 `tar -czvf`）的含义时。
- 当需要通过 `x man :关键词` 快速查看命令的社区参考（TLDR 风格）时。

## 核心原则与规范
- **交互式搜索**: 使用 `--fzf` 获得可搜索的 TUI 体验。
- **AI 解释**: 利用 `--explain` 标志将复杂的命令分解为可读的摘要。
- **混合支持**: 支持使用 `:` 前缀直接在 `man` 命令中访问 TLDR 内容。

## 实战示例

### 模糊搜索手册
```bash
# 从所有系统手册页中交互式选择
x man --fzf
```

### AI 命令解释
```bash
# 使用 AI 准确解释这些 tar 标志的作用
x man --explain "tar -czvf"
```

### 快速 TLDR 风格
```bash
# 查看 ssh 的简化 TLDR 参考
x man :ssh
```
