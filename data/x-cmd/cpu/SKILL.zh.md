# cpu - CPU 信息与检测

`cpu` 模块提供了一种快速获取系统处理器详细硬件信息并确定系统字节序的方法。

## 激活时机
- 当识别 CPU 架构或硬件规格时。
- 当为底层开发或网络编程检测系统字节序（小端 vs 大端）时。

## 核心原则与规范
- **简洁性**: 旨在进行快速的硬件检查。
- **字节序标志**: 小端返回 `l`，大端返回 `b`。

## 实战示例

### 查看 CPU 信息
```bash
# 显示详细的处理器硬件信息
x cpu info
```

### 检测字节序
```bash
# 检查系统字节序
x cpu endianness
```
