# path - PATH 变量管理

`path` 模块提供了一种简单有效的方式来操作 Shell 的 PATH 变量。它支持在路径中添加/删除目录，并对条目进行去重以保持环境清洁。

## 激活时机
- 当用户想要将新目录添加到 PATH 的开头 (`unshift`) 或末尾 (`push`) 时。
- 当从 PATH 中删除特定目录时。
- 当清理 PATH 变量中重复的目录条目时。
- 当在交互式应用中查看当前 PATH 结构时。

## 核心原则与规范
- **非破坏性**: 提供明确的子命令进行修改，而不是直接进行字符串编辑。
- **交互式 UI**: 使用默认的 `x path` 在结构化列表中浏览当前 PATH 条目。
- **优先级**: 注意路径是应该具有优先级（PATH 开头）还是作为回退（PATH 末尾）。

## 实战示例

### 添加到 PATH
```bash
# 将目录添加到 PATH 的开头
x path unshift /usr/local/custom/bin
```

### 移除重复项
```bash
# 移除 PATH 变量中所有多余的条目
x path uniq
```

### 查看与列出
```bash
# 列出 PATH 中当前引入的所有目录
x path ls
```
