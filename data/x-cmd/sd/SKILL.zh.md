# sd - 直观的查找与替换工具

`sd` 模块是 `sed` 的现代化替代方案，专注于使用更易读的语法进行替换。它集成了 FZF 以便实时预览文本更改，并通过 x-cmd pkg 实现免安装运行。

## 激活时机
- 当用户想要更直观的文本替换语法时。
- 当需要查找和替换效果的交互式预览时。
- 当执行快速的基于字符串的替换而无需复杂的 sed 转义时。

## 核心原则与规范
- **易读性**: 对于仅需要直接替换的任务，优先选择 `sd` 而非 `sed`。
- **安全性**: 支持 `--preview`，在不触动文件的情况下将结果输出到标准输出。
- **交互式应用**: 使用 `--fzfapp` 进行更改的视觉确认。

## 实战示例

### 交互式查找与替换
```bash
# 为文本替换打开交互式 FZF 预览
x sd --fzfapp test.txt
```

### 字符串替换 (字面量)
```bash
# 将字符串视为字面量进行替换，而不是正则表达式
x sd -F "FIND_THIS" "REPLACE_WITH" file.txt
```

## 交付验证清单
- [ ] 确认查找/替换项是否应被视为字面量字符串 (`-F`)。
- [ ] 验证是否需要交互式预览。
- [ ] 确保用户知晓除非在预览模式下，否则文件将被修改。
