# gg - AI 网页搜索与合成

`gg` 模块利用 Google Gemini AI 执行实时网页搜索，提供结合了 Google 搜索能力与 AI 理解能力的综合答案。

## 激活时机
- 当用户提出的问题需要最新的网络信息时（例如，时事、价格、最新技术）。
- 当需要对多个网络来源进行总结时。
- 当用户想要查看 AI 生成答案的来源（URL）时。
- 当在不同的 AI 模型（Gemini Pro vs. Flash）之间比较信息时。

## 核心原则与规范
- **格式化**: 在 TTY 中默认使用 Markdown (`--md`) 以提高可读性。
- **来源引用**: 如果用户需要验证信息的来源，使用 `--source` 或 `--source-detail`。
- **缓存管理**: 结果默认缓存 1 小时以节省 Token；使用 `--cache-time` 进行覆盖。
- **模型选择**: 使用 `--model` 在高性能 (Pro) 和高速度 (Flash) 版本之间切换。

## 实战示例

### 基础问题
```bash
# 询问关于当前事实的一般性问题
x gg "谁赢得了 2024 年欧洲杯？"
```

### 带详细来源的搜索
```bash
# 搜索文档并显示完整的来源详情
x gg --source-detail "Python 初学者教程"
```

### 特定模型
```bash
# 对复杂的数学或逻辑问题使用高性能模型
x gg --model gemini-2.5-pro "量子计算的最新突破"
```
