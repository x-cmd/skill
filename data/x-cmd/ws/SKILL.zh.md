# ws - 工作区管理工具

`ws` 模块提供了管理项目工作区的工具，允许在定义的工作区根目录内进行隔离的命令执行和文件访问。

## 激活时机
- 当用户想要识别当前工作区的根目录时。
- 当相对于工作区根目录执行命令或脚本文件时。
- 当在工作区环境中查看文件内容时。

## 核心原则与规范
- **隔离性**: 通过 `ws` 执行的命令具有对工作区根目录的上下文感知。
- **路径感知**: 在内部操作中使用相对于工作区 `.x-cmd` 目录的相对路径。

## 实战示例

### 显示工作区根目录
```bash
# 显示当前工作区的根目录
x ws --root
```

### 在工作区中执行
```bash
# 在工作区上下文中运行特定命令
x ws exec my-command
```
