# wkp - 维基百科搜索与摘录工具

`wkp` 模块允许用户搜索维基百科并从文章中提取文本内容。它支持交互式浏览、自动建议以及 DuckDuckGo 集成。

## 激活时机
- 当用户想要获取特定维基百科主题的快速摘要或摘录时。
- 当按标题搜索文章或针对拼写错误使用模糊建议时。
- 当想要从 CLI 直接在浏览器中打开维基百科页面时。

## 核心原则与规范
- **简洁性**: 优先使用 `extract` 或 `hop` 在终端中立即获取文本摘要。
- **搜索辅助**: 使用 `suggest` 帮助用户找到其查询的正确术语。
- **交互式 UI**: 使用 `--app` 模式探索搜索结果。

## 实战示例

### 获取摘要
```bash
# 获取 'OpenAI' 的简洁摘要
x wkp extract OpenAI
```

### 直接搜索与摘要
```bash
# 搜索 'Python' 并显示第一个结果的摘录
x wkp hop Python
```

### 搜索建议
```bash
# 获取维基百科对拼写错误的术语建议
x wkp suggest pythen
```
