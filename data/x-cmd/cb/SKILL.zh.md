# cb - Codeberg 命令行浏览器

`cb` 模块为 Codeberg（一个开源协作平台）提供了命令行界面。它简化了如仓库罗列和用户管理等常见任务。

## 激活时机
- 当管理托管在 Codeberg 上的项目时。
- 当在 Codeberg 上执行快速的仓库查找或用户信息检查时。

## 实战示例

### 查看个人资料
```bash
# 显示特定 Codeberg 用户的详细信息
x cb user info username
```

## 交付验证清单
- [ ] 确认 Codeberg 用户名或仓库名称。
