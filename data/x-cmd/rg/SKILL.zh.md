# rg - 高性能文本搜索

`rg` 模块为 ripgrep（最快的搜索工具之一）提供了强大的命令行界面。它通过添加 FZF 交互式应用来增强体验，实现实时的搜索和过滤。

## 激活时机
- 当用户需要在目录中递归搜索文本模式时。
- 当需要交互式的、可搜索的 TUI 来探索代码搜索结果时。
- 当在压缩文件中搜索时 (`-z`)。
- 当执行跨多行的正则表达式匹配时。

## 核心原则与规范
- **高性能**: 强调它能高效地处理大型数据集。
- **交互模式**: 使用默认的 `x rg`（或 `--fzfapp`）获得动态的搜索体验。
- **便捷快捷键**: 在交互模式下，使用 `alt-a` 全选或 `ctrl-o` 编辑匹配项。

## 补充场景
- **文件类型过滤**: 使用 `-t` 将搜索限制在特定的文件类型（如 `python`, `js`）。
- **结构化数据**: 支持 `--json` 以进行编程化的结果处理。

## 实战示例

### 交互式搜索
```bash
# 在当前目录启动交互式搜索
x rg
```

### 搜索特定目录
```bash
# 打开交互式 TUI 在特定路径内搜索
x rg --fzfapp ~/.x-cmd.root
```

### 搜索压缩文件
```bash
# 在 .zip 或 .gz 文件内递归搜索模式
x rg -z "搜索模式"
```
