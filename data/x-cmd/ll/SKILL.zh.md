# ll - 交互式资源列表

`ll` 模块提供了一个交互式界面（主要利用 `fzf`），用于搜索和列出有关文件系统以及关键操作系统资源（如 CPU、内存和进程）的信息。

## 激活时机
- 当用户想要使用 `fzf` 交互式地搜索文件或目录时。
- 当对系统 CPU、内存或进程状态执行快速交互式检查时。

## 核心原则与规范
- **交互式搜索**: 使用 `--fzfapp`（或默认的 `x ll`）搜索文件和系统数据。
- **资源快捷方式**: 支持前缀子命令（例如 `:cpu`, `:mem`）以获取快速的系统快照。

## 实战示例

### 交互式文件搜索
```bash
# 启动交互式 fzf 会话以浏览文件和目录
x ll
```

### 快速系统检查
```bash
# 通过 ll 查看当前内存或进程状态
x ll :mem
x ll :ps
```
