# ollama - 本地大模型管理与交互

`ollama` 模块为 Ollama 提供了增强的命令行界面，使用户能够轻松管理本地模型、执行翻译并与各种 AI 架构进行交互。

## 激活时机
- 当用户想要安装或管理 Ollama 服务时。
- 当从 Ollama 注册表拉取模型（如 Llama 3, Mistral）时。
- 当与本地模型聊天或将其用于基于文件的翻译时。
- 当使用交互式 UI 浏览和下载模型时。

## 核心原则与规范
- **注册表交互**: 使用 `pull` 和 `push` 与 Ollama 注册表同步模型。
- **快捷别名**: 支持使用 `@o` 别名进行快速的本地模型对话。
- **文件上下文**: 利用 `--file` 标志向模型提供本地上下文。

## 补充场景
- **交互式浏览**: 使用 `la` 通过交互式 UI 探索可用模型。
- **服务控制**: 使用 `serve` 手动启动 Ollama 后端。

## 实战示例

### 拉取并运行
```bash
# 下载并开始与 Mistral 聊天
x ollama pull mistral
@o "向量数据库是如何工作的？"
```

### 带有文件上下文的翻译
```bash
# 使用本地模型翻译本地文档
@o --file ./abstract.en.md "翻译为中文"
```

### 交互式 UI
```bash
# 以交互方式浏览 Ollama 注册表
x ollama la
```

## 交付验证清单
