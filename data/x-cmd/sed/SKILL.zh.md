# sed - 带有交互式预览的流编辑器

`sed` 模块通过添加交互式预览模式扩展了经典的流编辑器。用户可以实时测试其正则表达式替换，并在修改文件前查看上下文差异 (`ctrl-s`)。

## 激活时机
- 当用户想要执行文本查找和替换操作时。
- 当需要正则表达式效果的交互式预览以避免错误时。
- 当在文件中执行批量行删除或插入时。
- 当使用扩展正则表达式进行复杂的文本操作时。

## 核心原则与规范
- **预览先行**: 鼓励使用 `--fzfapp` 模式以直观地验证更改。
- **上下文差异**: 在交互式应用中使用 `ctrl-s` 查看 diff。
- **原地安全**: 谨慎使用 `-i`，最好带有备份后缀。

## 实战示例

### 交互式预览
```bash
# 在文件上交互式测试并预览正则更改
x sed --fzfapp test.txt
```

### 基础替换
```bash
# 在管道字符串中将 'world' 替换为 'universe'
echo "hello world" | x sed 's/world/universe/'
```

### 全局替换
```bash
# 全局替换文件中所有出现的 'foo' 为 'bar'
x sed 's/foo/bar/g' file.txt
```
