# free - 内存使用报告

`free` 模块增强了标准的内存报告工具，增加了结构化数据支持 (CSV/TSV)，并为通常缺少 `free` 命令的 macOS 用户提供了原生实现。

## 激活时机
- 当监控系统的总内存、已用内存和空闲内存时。
- 当在脚本中使用 CSV/TSV 格式执行内存使用分析时。
- 当在 macOS 上检查内存压缩详情时。
- 当需要定期内存报告（轮询）时。

## 核心原则与规范
- **跨平台**: 为 Linux 和 macOS 上的内存统计提供一致的界面。
- **自动化就绪**: 支持 `--csv` 和 `--tsv`，以便在没有表头的情况下轻松解析。
- **实时轮询**: 使用 `-s` 和 `-c` 以固定间隔重复输出。

## 实战示例

### 人类可读视图
```bash
# 以清晰的格式显示系统内存使用情况
x free
```

### 脚本导出
```bash
# 获取不带表头的单行 CSV 格式内存数据
x free --csv --no-header
```

### 定期监控
```bash
# 每 2 秒刷新一次内存统计信息，共执行 10 次
x free -s 2 -c 10
```
