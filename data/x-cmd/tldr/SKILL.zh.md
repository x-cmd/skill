# tldr - 简洁的命令备忘单

`tldr` 模块为数千种 CLI 工具提供简化的、社区驱动的文档。它不提供详尽的手册，而是专注于最常见的使用场景和实际示例。

## 激活时机
- 当用户想要快速了解如何使用特定命令时。
- 当寻求命令子任务（如 `git checkout`）的实际示例时。
- 当使用 `fzf` 交互式浏览命令参考时。

## 核心原则与规范
- **简洁性**: 强调这是为了“快速查找”，而不是详细研究。
- **语言支持**: 使用 `--lang` 以首选语言（如 `zh`）检索文档。
- **交互式浏览**: 支持使用 `tlfz` 快捷方式进行快速命令发现。

## 实战示例

### 快速参考
```bash
# 查看 'tar' 命令的常用使用示例
x tldr tar
```

### 子命令示例
```bash
# 获取 git checkout 的特定帮助
x tldr git checkout
```

### 交互式发现
```bash
# 使用 FZF 交互式搜索命令
x tldr --fz
```
