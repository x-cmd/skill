# hi - HTTP 信息摘要

`hi` 模块提供了一种快速查看当前 HTTP 元数据和请求信息的方法，通常用于调试连通性或从 HTTP 角度查看公网 IP 信息。

## 激活时机
- 当用户想要查看其面向公众的 HTTP 头和 IP 时。
- 当执行与 Web 请求相关的基础网络诊断检查时。

## 实战示例

### 查看 HTTP 信息
```bash
# 显示常规 HTTP 环境和连接详情
x hi
```
