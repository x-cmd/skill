# ip - IP 地址与地理位置工具

`ip` 模块提供了一套全面的工具，用于识别 IP 地址及其地理位置，并执行基础的网络侦察，如子网映射和端口扫描。

## 激活时机
- 当用户想知道其本地或公网 IP 地址时。
- 当检索特定 IP 的地理位置数据（国家、城市、ISP）时。
- 当在 CIDR 范围内识别活跃主机时。
- 当对目标执行快速的 TCP 端口扫描时。

## 核心原则与规范
- **外部数据**: 为地理位置子命令使用 Geolite (ipinfo.io)。
- **网络侦察**: 支持使用 `map` 发现活跃 IP，使用 `tps` 进行端口扫描。
- **平台无关**: 在不同的操作系统环境中为 `ifconfig` 和 `ip addr` 提供统一的界面。

## 实战示例

### 本地 IP 信息
```bash
# 列出所有本地网络接口的 IP 地址
x ip ls
```

### 地理位置
```bash
# 获取公网 IP（如 8.8.8.8）的地理位置
x ip geolite 8.8.8.8
```

### 网络映射
```bash
# 在特定子网中发现活跃主机
x ip map 192.168.1.0/24
```
