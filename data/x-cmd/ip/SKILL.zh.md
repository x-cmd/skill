# x ip - 网络诊断与地理查询 (AI 优化版)

`x ip` 模块提供了简洁的接口来查询 IP 地址相关信息，无论是本机网卡配置还是全球 IP 的地理归属。

## 激活时机
- 当需要获取本机公网 IP 或内网网卡地址时。
- 当需要查询特定 IP 地址的地理位置、所属 ISP 等信息时。
- 当需要扫描子网中的活跃主机或开放端口时（网络诊断）。

## 核心原则与规范
- **非交互优先**: 避免使用交互式配置，直接通过子命令获取数据。
- **结构化思维**: 部分子命令（如 `geolite`）提供详细的地理信息。

## 实战示例

### 查询本机公网 IP 及地理位置
```bash
# 获取当前公网出口 IP 的详细地理信息
x ip geolite
```

### 查询特定 IP 的归属地
```bash
# 查询 8.8.8.8 的详细位置信息
x ip geolite 8.8.8.8
```

### 列出所有本地网卡地址
```bash
# 获取本机所有 IP 地址列表（非交互）
x ip ls
```

### 网络扫描 (诊断任务)
```bash
# 扫描子网内的活跃主机 (需要权限)
x ip map 192.168.1.0/24

# 快速扫描特定 IP 的常见端口
x ip tps 127.0.0.1
```

## 交付验证清单
