# ffmpeg - 全能多媒体处理器

`ffmpeg` 模块提供了一个强大的命令行界面，用于处理音频、视频和其他多媒体文件。它通过在系统缺失 ffmpeg 时利用 pixi 自动安装，确保了工具的可用性。

## 激活时机
- 当在不同的视频或音频格式之间进行转换时（例如 MP4 转 AVI）。
- 当从视频文件中提取音频轨道时（例如 MP4 转 MP3）。
- 当执行复杂的媒体操作（如调整大小、重新编码或应用过滤器）时。
- 当需要从终端流式传输多媒体内容时。

## 核心原则与规范
- **零设置**: 如有必要，会自动通过 `pixi` 处理安装。
- **透明透传**: 通过 `--` 或 `cmd` 子命令支持所有标准的 `ffmpeg` 参数。
- **输入/输出**: 始终明确 `-i`（输入）和目标输出路径。

## 实战示例

### 格式转换
```bash
# 将视频从 mp4 转换为 avi 格式
x ffmpeg -i input.mp4 output.avi
```

### 提取音频
```bash
# 从视频文件中提取音频并保存为 MP3
x ffmpeg -i input.mp4 -vn output.mp3
```

### 直接执行
```bash
# 执行原始的 ffmpeg 命令字符串
x ffmpeg --cmd -version
```
