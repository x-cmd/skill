# asdf - 多版本运行环境管理器

`asdf` 模块通过提供简化的 x-cmd 风格命令来安装和切换环境运行时，增强了标准的 asdf 版本管理器。

## 激活时机
- 当用户想要安装新的编程语言环境（如 Node.js, Elixir）时。
- 当全局或按项目在工具的不同版本之间进行切换时。
- 当管理 asdf 插件或更新 asdf 核心时。

## 核心原则与规范
- **增强流**: 使用 `use` 自动添加插件、安装最新版本并将其设置为全局版本。
- **环境状态**: 使用 `--activate` 和 `--deactivate` 控制 asdf 管理的二进制文件是否处于当前 Shell 的 PATH 中。
- **零设置**: 如果 asdf 缺失，会自动处理其安装。

## 实战示例

### 安装并使用
```bash
# 安装最新的 Node.js 并将其设置为全局版本
x asdf use nodejs
```

### 受控卸载
```bash
# 卸载运行时并移除其对应的 asdf 插件
x asdf unuse nodejs
```

### 搜索插件
```bash
# 交互式地搜索可安装的软件/插件
x asdf
```
